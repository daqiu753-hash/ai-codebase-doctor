# Field Tests

This directory is for recording real-project trial runs of `ai-codebase-doctor`.

Field tests should help answer:

- Which findings were useful?
- Which findings were false positives?
- Which obvious repo reality issues were missed?
- Which scanner improvements would reduce friction?

Do not include private source code, secrets, `.env` values, customer data, or proprietary logs. Prefer public repositories, generated demo repos, or summarized observations.

Use [field-test-template.md](field-test-template.md) for each trial run.
