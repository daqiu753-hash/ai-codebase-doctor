# Code of Conduct

This project follows a simple standard: be respectful, concrete, and useful.

## Expected Behavior

- Be kind and professional.
- Assume good intent.
- Keep feedback actionable.
- Respect project scope and maintainer time.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Sharing private data, secrets, or credentials.
- Spam or off-topic promotion.

Maintainers may remove content or participation that harms the project community.
